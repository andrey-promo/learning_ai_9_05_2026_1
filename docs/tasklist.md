## Итерации

### Итерация 1 — Скелет, конфиг, живой бот
> Цель: бот запускается, читает конфиг и отвечает на команды.

- [x] Создать `pyproject.toml` с зависимостями (aiogram, openai, python-dotenv)
- [x] Создать структуру `src/telegram_llm_bot/` с пустыми модулями
- [x] Создать `.env.example` с перечнем всех переменных
- [x] Реализовать `config.py` — загрузка и валидация env, логирование
- [x] Реализовать `main.py` — инициализация Bot, Dispatcher, запуск polling
- [x] Реализовать `handlers.py` — `/start`, `/help`, заглушка на нетекст
- [x] Убедиться: `/start` возвращает ответ в личном чате

### Итерация 2 — Подключение LLM (без истории)
> Цель: бот отвечает на текст через OpenRouter, каждый запрос независим.

- [x] Реализовать `llm_client.py` — обёртка над OpenAI-клиентом с `base_url` OpenRouter
- [x] Подключить `llm_client` в хендлер текстовых сообщений
- [x] Убедиться: пользователь пишет текст → получает ответ от модели
